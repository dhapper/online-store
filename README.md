# Spring Boot E-commerce Project

A simple e-commerce application built with Spring Boot.

## Features

*   Product display and entry page.

## Technologies

*   **Backend:**
    *   Spring Boot
    *   Java
    *   Maven (Dependency Management)
    *   PostgresSQL (Supabase)
*   **Frontend:**
    *   HTML
    *   CSS
    *   JavaScript (For client-side scripting)
